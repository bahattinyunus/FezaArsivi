# Katkıda Bulunma

Katkılarınızı memnuniyetle karşılıyoruz. Lütfen aşağıdaki adımları izleyin:

1. Reponu forkla.
2. Yeni proje için uygun klasör oluşturun veya mevcut bir dosyayı güncelleyin.
3. Değişiklikleri anlamlı commit mesajları ile gönderin.
4. Pull Request açın ve yaptığınız değişiklikleri açıklayın.

Ayrıntılar eklenecek.
